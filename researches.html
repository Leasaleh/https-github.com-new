CSS Rules

CSS rules has two main parts: -Selectors: It determine which element the HTML applies to. There are many types of selectors, we have the Universal Selector that targets all the elements on the screen. The Type Selector that targets specific HTML tags. Class Selector targets element ith specific class attribute. ID Selector targets elements wit specific ID that starts with a #. Descendant Selector that applies the HTML to all the children of a parent. Pseudo-classes that applies HTML to a state of an element. -Declarations block: It specifies a CSS property and value. We have many types of declarations like the Color and Background, Font and Text, Box Model (Margin, Padding, Border), Position of element (Width, Height), Responsive Design that adjust the styles tothe screen size.[16]
