Importance of the security

The security on the web is important to protect ourselves from online threats. To protect websites and web application from hackers, data breaches, and other bad intentions a lot of practices and technologies are required. With the growth of cyber threats, the web security is constantly working on evolving new ways and technologies to protect people's informations and privacy.[6]

Common Web Security Threats

Cuber threat is a form of online threat used to harm people online. There is many way to execute these threats but the most common ways are:

  • Phishing: which is usually done through emails and messages it is usually done through tricking the victims into trusting them and giving them personal informations such as phone number, credit cart data,...
  • Ransomware: It is a form of software that uses encryption to prevent users to access their data, usually they will hold the data hostage as long as the ransom is not paid
  • SQL injection: where attackers use the vulnerabilities in the application code to extract informations from the user.
  • Cross-Site Scripting: it is a type of injection where scripts are injected in trusted web that helps the hacker to access personal data
  • Distributed Denial-of-Service (DDos) Attack: It is one of the most powerful weapon on the cyber platform.. It uses multiple servers and Internet connections to flood targeted sources by crating heavy traffic.
  • Viruses and worms: these programs spread across our networks and computer and steal data from the users.[7]
  • Spyware: it is a type of malicious software (malware) that secretly monitors and collects information without the user knowing such as passwords

Safe Practices

To avoid all kind of cyber threats, some actions are required, it is important not to use public Wi-Fi when doing banking or online shoping that transmit some sensitive informations. Moreover, strong passwords are required for more protection. A strong password should be long with a mix of characters. Enabling multi-factor authentification is also a great step to protect ourselves from online traps, it coulsd be an extra password or some personal security questions or some fingerprints. It is also important not enter any suspicious websites or open any untrusted links or download anything suspitious that couls help hackers enter personal informations.[8]
